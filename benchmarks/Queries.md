The following Queries have been used to test the performance of LLM's for different tasks.

### Token generation

```
Schreibe den Buchstaben "A" 5000 Mal hintereinander ohne Leerzeichen.
```

### Text generation

```
Erkläre in einfachen Worten, wie ein neuronales Netz funktioniert, und gib dabei etwa 1000 Wörter aus.
```

### Complex reasoning

```
Löse folgende Aufgabe Schritt für Schritt und erkläre jeden Schritt ausführlich:
Ein Zug fährt mit 80 km/h von Berlin nach Hamburg (ca. 300 km). 30 Minuten nach der Abfahrt startet ein zweiter Zug mit 120 km/h in dieselbe Richtung. Wann und wo überholt der zweite Zug den ersten?
```

### Summarize a large context

```
Hier ist ein sehr langer Text. (Hier Inhalt von https://de.wikipedia.org/wiki/Quantencomputer einfügen)
Fasse den Text in 200 Wörtern zusammen.
```
